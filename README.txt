Подготовка
1. Убедитесь, что у вас установлен Node.js (в интернете много информации по установке).
2. В терминале перейдите в директорию с этим проектом и выполните команду npm install. Эта команда установит все необходимые зависимости для работы проекта.
3. Через проводник вы можете изменить комментарии, которые будут оставляться. Для этого откройте файл comments.txt и напишите свои комментарии. Важно: разделяйте комментарии знаком $ (как в примере файла). Количество комментариев может быть любым.
4. Проект готов к запуску.

Начало работы
1. В терминале перейдите в директорию с проектом (если вы ещё не там) и выполните команду npm start. Скрипт запустится, и откроется окно логина в TikTok.
2. Перейдите на страницу Feedback and help и войдите в аккаунт оттуда (красная кнопка Log in). Это позволяет обойти защиту TikTok от ботов. После входа закройте вкладку и перезапустите скрипт.
3. После перезапуска вы автоматически войдёте в аккаунт, и скрипт начнёт свою работу. Под видео будет случайным образом оставлен один из комментариев, которые вы указали в файле comments.txt.
4. Чтобы остановить скрипт, закройте окно браузера или консоль. Скрипт также автоматически завершит работу через 12 часов.

Важно знать
1. Комментарий оставляется каждые 15 секунд.
2. Разделяйте комментарии в файле comments.txt знаком $ (как в примере файла).
3. Во время работы скрипта не кликайте мышкой по открытому окну браузера, это нарушит работу скрипта. В противном случае потребуется его перезапуск. Лучше просто сверните окно браузера и консоль.
4. Убедитесь, что TikTok открыт на английском языке (URL должен быть: https://www.tiktok.com/foryou?lang=en).

Удачи :)

Preparation
1. Make sure you have Node.js installed (there is plenty of information on the Internet about how to install it).
2. In the terminal, go to the directory with this project and run the npm install command. This command will install all the necessary dependencies for the project to work.
3. You can change the comments that will be left using File Explorer. To do this, open the comments.txt file and write your comments. Important: separate comments with the $ sign (as in the file example). You can leave as many comments as you like.
4. The project is ready to run.

Getting started
1. In the terminal, navigate to the project directory (if you are not already there) and run the npm start command. The script will start, and the TikTok login window will open.
2. Go to the Feedback and help page and log in to your account from there (red Log in button). This allows you to bypass TikTok's bot protection. After logging in, close the tab and restart the script.
3. After restarting, you will automatically log into your account, and the script will start working. One of the comments you specified in the comments.txt file will be randomly left under the video.
4. To stop the script, close the browser window or console. The script will also automatically stop working after 12 hours.

Important to know
1. Comments are posted every 15 seconds.
2. Separate comments in the comments.txt file with the $ sign (as in the file example).
3. While the script is running, do not click on the open browser window, as this will disrupt the script's operation. Otherwise, you will need to restart it. It is better to simply minimize the browser window and console.
4. Make sure that TikTok is open in English (the URL should be: https://www.tiktok.com/foryou?lang=en).

Good luck :)
